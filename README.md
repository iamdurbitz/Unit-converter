A basic unit converter app. This project is a Metric/Imperial Unit Converter that allows users to convert between:  Meters ↔ Feet, Liters ↔ Gallons, Kilograms ↔ Pounds. What I practiced in this project:  JavaScript fundamentals (variables, functions, calculations), DOM manipulation, Handling user input , Dynamic rendering of results, Clean and readable code structure. Goal: The main goal of this project was to build a fully functional interactive app using JavaScript, rather than focusing only on design.
